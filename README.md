# Server Log Parser

This is a program which parses web server logs 
## Setup
- Install Ruby 2.7.0

- Install Bundler

  `$ gem install bundler`

### Install gems

  `$ bundle install`

## Running

  `$ ruby parser.rb webserver.log`

## Testing

You can test parser using Rspec.

  `$ rspec`
